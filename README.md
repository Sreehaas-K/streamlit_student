# Student Performance Management System

A basic Streamlit application to manage student performance data using MySQL.

## Features
- Add, view, update, and delete student records
- Pass/Fail status
- Average marks and pass percentage
- Subject-wise average and top scorer

## Tech Stack
- Streamlit
- Python
- MySQL
- Pandas

## How to Run
1. Execute `db.sql` in MySQL
2. Install dependencies: `pip install -r requirements.txt`
3. Run app: `streamlit run app.py`
